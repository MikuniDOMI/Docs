# 启动游戏

貌似是一个启动核心最有用的地方（？

## 先前条件

:::warning
你需要先读取游戏，才能进行启动游戏！
:::  

## 使用

```C
ClientRunner Runner = new ClientRunner(new ClientRunnerInfo()
{
    GameInstances = ver,
    JavaInfo = new JavaInfo()
    {
        JavaPath = @"D:\\MCLDownload\\ext\\jre-v64-220420\\jdk17\\bin\\java.exe", // Java 地址，javaw 和 java 都可。
    },
    Account = new OffineAuthenticator("RoundStudio").Authenticator(), // 离线账户
    LauncherInfo = "启动器水印标识",
    LauncherVersion = "启动器版本号",
    WindowInfo = ClientWindowSizeEnum.Fullscreen
});
Runner.LogsOutput = (string logs) => { Console.WriteLine(logs); }; // 日志输出回调
Runner.Start();
```
